## GDELT methodology and applications

**GDELT Project — "Television as Data: Opening TV News for Deep Analysis and New Forms of Interactive Search" (2022).** Library as Laboratory series, session 4, Internet Archive.  
Session recap: https://blog.archive.org/2022/04/25/library-as-laboratory-opening-television-news-for-deep-analysis-and-new-forms-of-interactive-search/  
Series overview: https://blog.archive.org/2022/02/09/library-as-laboratory-a-new-series-exploring-the-computational-use-of-internet-archive-collections/

Specific case studies referenced in this session, all linked from the resource guide above:
- Visualizing entity graphs (co-occurrence networks, including the Mars / Elon Musk case)
- Using Google's Cloud Inference API to contextualize entities across the news
- Constructing a chronology of doctors on television news
- Tracking CNN's COVID-19 dashboard through onscreen text search
- Comparing human and machine transcription using speech recognition
- Full dataset of Donald Trump tweets on television news
- All GDELT datasets: https://www.gdeltproject.org/data.html
